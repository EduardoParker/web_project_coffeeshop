# Triple Espresso

Nombre del proyecto - Web Triple Espresso
descripcion: en este proyecto, podemos encontrar una pagina basica de la
cafeteria triple espesso, en donde podemos ver informacion basica de la localizacion de establecimiento y sus horarios, en donde se usaron texnicas de flex box, listas y posicionameinto.
En la segunda seccion encontramos unos videos explicando recestas de como realizar un bune cafe, en esta seccion aplicamos conocimientos de embeded code para los videos de youtube.
En la tercera encontramos un apartado en donde puedes realizar una reservacion, para esto usamos formularios y su modificacion en CSS y pseudoelementos.
el la ultima parte vemos el calisica pie de pagina, en donde aplicacomo un overflow, flex box y posicionamiento.

una mejora logica, es la aplicacion de javascrips para el correcto funcionamiento de los form
